# CSV Collaboration Dashboard

A full-stack web application that allows users to upload CSV datasets, manage records, search data, and collaborate in real-time through conflict detection.

---

## Features

### CSV Upload
- Upload CSV files through the web interface
- Validate CSV data before saving
- Bulk insert records into MySQL
- Upload progress indicator

### Record Management
- View uploaded records
- Pagination support
- Search by name, email, or content

---

## Tech Stack

### Frontend
- React

### Backend
- Node.js
- Express.js

- CSV Parser
